# Quiz Management System

A dynamic **Quiz Management System** built using **HTML, CSS, and JavaScript** that allows examiners to create and manage timed quizzes while providing a seamless experience for participants.

## 🚀 Features

### Examiner Features
- Create and manage multiple quizzes
- Add, edit, and delete quiz questions
- Set quiz duration with countdown timer
- Configure multiple-choice questions
- Organize questions efficiently

### Participant Features
- Attempt quizzes through an intuitive interface
- Real-time countdown timer
- Automatic response saving
- Navigate between questions easily
- Instant quiz submission upon completion

### System Features
- ⏱️ Timed quiz functionality
- 💾 Auto-save responses to prevent data loss
- 📤 Automated submission when time expires
- 📱 Responsive and user-friendly design
- ⚡ Fast and lightweight implementation
- 🔄 Dynamic question rendering using JavaScript

## 🛠️ Technologies Used

- **HTML5** – Structure and content
- **CSS3** – Styling and responsive design
- **JavaScript (ES6)** – Quiz logic, timer, auto-save, and submission handling
