## A dynamic Quiz Management System built with HTML, CSS, and JavaScript that enables examiners to create timed quizzes with multiple questions, auto-save responses, and automated submission functionality.

keyFeatures: [
  "HTML5",
  "CSS3",
  "JavaScript (ES6)",
  "DOM Manipulation",
  "Local Storage",
  "Dynamic Question Creation",
  "Multiple Choice Question Management",
  "Correct Answer Configuration",
  "Marks Assignment",
  "Quiz Duration and Countdown Timer",
  "Auto Save Functionality",
  "Automatic Submission on Timeout",
  "Quiz Result Calculation",
  "Form Validation",
  "Responsive User Interface"
]
